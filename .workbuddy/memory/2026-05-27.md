## 发音功能重写
- 原代码问题：SpeechSynthesis 和 Google TTS 双重调用链、回调嵌套导致静默失败
- 重写为 4 级 async/await 降级链：
  1. Web Speech API (Promise封装，2秒超时自动降级)
  2. 有道词典 MP3 (`dict.youdao.com/dictvoice`，国内直连)
  3. Sound of Text 免费 API (异步轮询获取音频)
  4. SpeechSynthesis 兜底 (无 voice 过滤)
- 删除了底部的 `<audio id="tts-audio">` 固定标签，改为函数内动态创建
- 已推送 GitHub

## 修复列表不刷新
- 根因：`renderList()` 首次有卡片时 `el.innerHTML=''` 把 `#add-empty` 从 DOM 移除，后续 `document.getElementById('add-empty')` 返回 null 导致函数静默失败
- 修复：改为 `innerHTML` 直接重建空提示，不再依赖固定 DOM 元素
- 去掉了 `play()` 中的"正在播放..."toast
- 去掉了 `confirmImageSelection`/`confirmDelete` 中不必要的 setTimeout
- 已 commit 本地，GitHub 网络不通待推送
