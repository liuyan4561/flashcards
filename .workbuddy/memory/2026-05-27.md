# 2026-05-27

- 在闪卡列表每项添加了编辑按钮（✎），放在删除按钮前面
- 新增编辑弹窗（edit-dialog），支持修改单词、释义和图片
- 图片编辑复用现有图片选择弹窗，区分编辑模式/添加模式（通过 editId 判断）
- Store 新增 updateCard(id, word, hint) 方法
- 编辑模式下点击"更换图片"会预加载当前已有的图片到图片选择弹窗
- 支持删除单张已有图片（× 按钮）、搜索换一批、添加本地图片

- **中文释义与提示语彻底分离**：
  - 卡片数据模型新增 `meaning`（中文释义）字段，与 `hint`（英文句子提示）分离
  - 添加弹窗：提示语改为「英文句子」输入框；新增「中文释义」输入框，失焦时自动调用 Google Translate API 查询中文
  - 编辑弹窗：新增中文释义编辑字段
  - 学习页查看答案时展示顺序：图片 → 中文释义 → 英文提示句子
  - 批量导入：创建卡片后自动批量查询中文释义（autoFetchMeanings）
  - 查询 API 优先级：Google Translate → MyMemory → FreeDictionary（兜底英文释义）
