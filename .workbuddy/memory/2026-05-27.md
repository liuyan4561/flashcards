# 2026-05-27 工作日志

## 发音功能重写
- 原代码问题：SpeechSynthesis 和 Google TTS 双重调用链、回调嵌套导致静默失败
- 重写为 4 级 async/await 降级链：
  1. Web Speech API (Promise封装，2秒超时自动降级)
  2. 有道词典 MP3 (`dict.youdao.com/dictvoice`，国内直连)
  3. Sound of Text 免费 API (异步轮询获取音频)
  4. SpeechSynthesis 兜底 (无 voice 过滤)
- 删除了底部的 `<audio id="tts-audio">` 固定标签，改为函数内动态创建
- 已推送 GitHub
